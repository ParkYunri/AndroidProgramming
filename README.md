# Android Programming
#### final project
1. 프로젝트는 개인적인 아이디어를 배운 위젯 등을 사용하여 앱을 만든다.
 
2.  만든 앱의 초기화면에서 버튼을 누르면, 인텐트를 사용하여 새로운 화면(액티비티)을 만들어서 연결한 후, 새로운 화면에 다음 내용을 보여준다.   
 (1) 이름, 학과, 취미 등을 텍스트와 이미지 등으로 알려준다.   
 (2) 아날로그시계와 디지탈 시계를 보여준다.   
 (3) 자신의 일기장을 만든다.   
 (4) 본인이 좋아하는 노래 한곡을 들을 수 있게 한다.
